    Terminal的配置👻

    • Oh My Zsh的配置

      • 安装Oh My Zsh

      • 配置Oh My Zsh的主题

        Text Only
        ZSH_THEME="agnoster"

      • 配置Oh My Zsh的插件

        Text Only
        plugins=(
    git
    z
    zsh-autosuggestions
    zsh-syntax-highlighting    
)

        git和z是自带的插件。

        zsh-autosuggestions和zsh-syntax-highlighting是第三方插件需要自行安装，安装方法：

        对于zsh-autosuggestions

        Bash
        git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

        对于zsh-syntax-highlighting

        Bash
        git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

    • 设置终端字体和主题

      • 字体链接

        MonacoForPowerline.ttf，将其添加到字体册中

      • 主题链接

        Dracula

      • 设置终端的描述文件

        点击左下角...，导入下载好的 Dracula.teminal，更改字体

      • 终端呈现的样式

    Homebrew的安装👻

    • 参考文档

      • 设置为清华源

        Bash
        export HOMEBREW_INSTALL_FROM_API=1
export HOMEBREW_API_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles/api"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles"
export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"
export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"

      • 按顺序运行命令

        • 下载安装脚本

          Bash
          git clone --depth=1 https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/install.git brew-install

        • 执行安装

          Bash
          /bin/bash brew-install/install.sh

        • 安装完成后配置环境

          Bash
          (echo; echo 'eval "$(/usr/local/bin/brew shellenv)"') >> /Users/coco/.zprofile
eval "$(/usr/local/bin/brew shellenv)"

        • 删除安装包

          Bash
          rm -rf brew-install

    • 日常使用，可设置成清华源镜像

      Bash
      export HOMEBREW_API_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles/api"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles"
export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"
export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"
export HOMEBREW_PIP_INDEX_URL="https://pypi.tuna.tsinghua.edu.cn/simple"

    VScode的安装👻

    • 使用homebrew安装

      Bash
      brew install --cask visual-studio-code

    • 偏好设置

      • 插件

        主题：Dracula Official

        图标：vscode-icons-mac

        点右上方漏斗一样的图标，找到 Modified，可以查看修改的设置

        User的 ==settings.json==文件

        JSON
        {
    "workbench.colorTheme": "Dracula",
    "workbench.iconTheme": "vscode-icons-mac",
    "editor.wordWrap": "on"
}

      • 自定义快捷键

        新建文件：New File

        新建文件夹 ：New Folder

        新建窗口：New Window

        注：点击右上角...，找到 show User keybindings，可以查看修改的快捷键

        User的 keybindings.json 文件

        JSON
        // Place your key bindings in this file to override the defaults
[
    {
        "key": "cmd+n",
        "command": "explorer.newFile"
    },
    {
        "key": "shift+cmd+n",
        "command": "explorer.newFolder"
    },
    {
        "key": "shift+alt+cmd+n",
        "command": "workbench.action.newWindow"
    },
    {
        "key": "shift+cmd+n",
        "command": "-workbench.action.newWindow"
    }
]

    Royal tsx的安装👻

    • 使用homebrew安装

      Bash
      brew install --cask royal-tsx

    • Royal tsx的配置

      1. 安装两个必备插件

        • Terminal
        • File Transfer

      2. 新建一个Document（命名为server_config）

        • 在Credentials中，找到Add Credential

          • 在 Credential->Credential 中设置服务器中个人账号和密码。

        • 在Connections中新建一个文件夹（命名为Server-2080ti），找到Add Terminal设置SSH

          • 在 Teriminal->Terminal 中设置服务器的IP

          • 在 Common->Credentials 中设置之前已经创建好的账户

        • 在Connections中，找到Add File Transfer设置SFTP

          • 在 File Transfer->File Transfer 中设置服务器的IP

          • 在 Common->Credentials 中设置之前已经创建好的账户
